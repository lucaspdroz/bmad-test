# Requirements

## Functional Requirements

### Cards
- Create card with text
- Edit card text
- Delete card
- Tap card → triggers speech

### Folders (Groups)
- Create folder
- Rename folder
- Delete folder
- Assign cards to folders

### Drag and Drop
- Move cards within a folder
- Move cards between folders
- Reorder folders

### Speech
- Convert text to speech
- Use device native TTS
- Instant playback on tap

---

## Non-Functional Requirements

- Large buttons (accessibility)
- High contrast UI
- Offline-first
- Fast response (no lag on tap)
- Simple UX (low cognitive load)

---

## Constraints

- MVP must be simple
- No backend initially
- Store data locally
